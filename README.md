# 老司机出行助手

基于微信小程序平台，我们选择设计一款面对出租车司机的出行助手软件，主要实现了 司机载客数统计、加油数统计、活动区域统计、城市载客活跃点分析、司机载客数量 排名等功能，致力于解决以上我们提出的问题。以手机定位的功能进行辅助，可以查 询附近载客点以及兴趣点，给司机提供方便。
